## 如何使用

1. 按顺序完成每个练习
2. 先看 `题目.md` 了解要求
3. 在对应的 `.rb` 文件中编写代码
4. 运行 `ruby 文件名.rb` 测试
5. 如果有测试文件,运行 `ruby 文件名_test.rb`
6. 完成后查看 `solution.rb` 参考答案

## 练习提示

- 每个练习都有多种解法,先实现功能,再优化
- 遇到困难先自己思考 15 分钟,再查资料
- 完成后思考是否有更简洁的写法
- 记录学习笔记和心得
